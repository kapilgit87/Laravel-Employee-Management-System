# Laravel Employee Management System

A **modern Employee Management application** built with Laravel and Bootstrap.  
Manage employee records efficiently with profile image upload and a clean dashboard.

## Features

- Add, View, Edit, and Delete Employees
- Profile image upload and display
- Responsive UI using Bootstrap 5
- Modern dashboard with cards and hover effects
- Beginner-friendly and scalable for larger projects
